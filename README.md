# Get Info Website

Landing site for the Get Info app: a location-based community help platform where people ask real-world questions and get matched with nearby helpers. The website highlights the product, safety approach, and core features.

## Live Site (GitHub Pages)
```
https://teut2711.github.io/get_info_app/
```

## Run Locally
From `website/`:
```sh
npm i
npm run dev
```

## Build
```sh
npm run build
```

## Tech Stack
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui
